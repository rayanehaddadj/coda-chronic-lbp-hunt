
# Reallocations between 24-hour movement behaviours and risk of chronic low back pain: The HUNT Study

The repository includes relevant documentation and code to replicate the
analysis conducted in the paper:

**Add paper and doi**

The aim of this paper was to investigate the association between reallocating time 
from 24/7 movements behaviours and the risk of chronic low back pain. To respond 
to this question, we use data from the HUNT4 (2017-2019) and HUNT COVID (2021-2023)
studies.
